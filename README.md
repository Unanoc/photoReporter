# photoReporter

### How to use
Download it!
```
git clone https://github.com/Unanoc/photoReporter.git
```
Build it!
```
go build .
```
Execute it!
```
./photoReporter <flag> <src directory> <dest directory>
```
flags:
```
-u <path to dir with photo> <path to result dir>
-all <path to dir with dirs(with photos)> <path to result dir>
```
